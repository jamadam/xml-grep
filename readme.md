mojo-checkbot 0.01 beta
---------------

## SYNOPSIS
    
    xml-grep 'DIR' 'CSS Selector'

## DESCRIPTION

CSSセレクターでgrepします。「Mojo::DOM」を
ベースにしていますので、

[Mojo::DOM]
[Mojo::DOM]:http://search.cpan.org/~sri/Mojolicious/lib/Mojo/DOM.pm

Perl v5.10.1以上が必要です。もし古い場合は、アップグレードするか、perlbrewでググってください。

### EXAMPLE

    $ xml−grep ./ 'a[href]'
    $ xml−grep ./ 'tbody th'
